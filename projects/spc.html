This is a continuation/update of sijobe's spc mod.

#Installation

  1. install forge
  2. download the mod jar (in the downloads section)
  3. put the jar in the mods folder
    a) if you're on windows open the file explorer
    b) type in "%appdata%" in the top box
    c) go into the "mods" folder
    d) drag or copy the mod jar into the mods folder
